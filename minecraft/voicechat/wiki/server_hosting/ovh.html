Setup Guide for OVH

Submitted by Double0negative

This might only apply to servers sold as GAME

Setup Instructions

  • Add the voice chat port to the GAME Firewall
  • In the OVH control panel, click IP on the left menu
  • Find the IP that matches your server
  • Click the three dot menu on the right, and click Configure the GAME Firewall
  • From there, you can add the voice chat UDP port

Last update on 29.03.2023. Visit the hoster website 🡕
